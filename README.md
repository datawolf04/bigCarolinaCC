This repository takes results from [MileSplit.com](https://nc.milesplit.com/) and collects the Cross Country Results for the teams in the Big Carolina Conference 6A/7A within NCHSAA. These are:

- J.H. Rose (6A)
- Jacksonville (6A)
- White Oak (6A)
- D.H. Conley (7A)
- New Bern (7A)
- South Central (7A)
